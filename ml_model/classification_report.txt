FINAL MODEL — Arsitektur Baru
================================
Pipeline: ug/m3 -> noise(ug/m3) -> ISPU conversion -> feature engineering -> ML
ML sebagai robustness layer, BUKAN pengganti ISPU.

Arsitektur final:
  Sensor ug/m3
    |-> noise simulation (training only)
  ISPU conversion (deterministik, rule-based - WAJIB)
    |-> feature engineering (9 fitur)
  Ensemble ML -> pilih terbaik via drift test
    |-> Output: class + probability + risk_score

9 Features (dari ISPU index 0-500):
  1. max_ratio      - dominance of dominant pollutant
  2. gap_12         - how much dominant exceeds 2nd
  3. pm10_over_pm25 - dust vs combustion proxy
  4. max_over_mean   - instability indicator
  5. pm25_times_no2 - traffic combustion proxy
  6. weighted_hazard - weighted sum (PM2.5*1 + PM10*0.6 + O3*0.4 + NO2*0.3 + CO*0.1)
  7. entropy        - diversity of pollution sources
  8. skew           - distribution asymmetry
  9. std            - raw spread
